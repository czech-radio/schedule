Recipe: Get schedules for a given period and given station

Problem

You want to get a schedules for a given period and given station.

Solution

There si a methods Schedule::get_any_schedule(since, till) wich returns a collection of schedules for the given period of specified station.

[1]:

import pandas as pd
import datetime as dt
from cro.schedule import Client

Create the client instance with specified station ID.

[2]:

client = Client(sid="plus")

Get week schedules – one for the each day of the week.

[3]:

since_date, till_date = "2022-01-01", "2022-01-15"
# Both `date = '2022-01-31'` and `dt.date(2022, 1, 31)` are valid date formats.

schedules = client.get_any_schedule(since=since_date, till=till_date)

See the results.

[4]:

print(
    f"Fetched {len(schedules)} schedules for dates {[schedule.date.isoformat() for schedule in schedules]}."
)

Fetched 15 schedules for dates ['2022-01-01', '2022-01-02', '2022-01-03', '2022-01-04', '2022-01-05', '2022-01-06', '2022-01-07', '2022-01-08', '2022-01-09', '2022-01-10', '2022-01-11', '2022-01-12', '2022-01-13', '2022-01-14', '2022-01-15'].

You can convert each schedule do pandas.DataFrame and write the result for a whole week to the Excel.

[5]:

schedule_dfs = [schedule.to_table() for schedule in schedules]

with pd.ExcelWriter(
    f"../../../data/sheet/Schedule_P{since_date}_{till_date}.xlsx"
) as writer:
    pd.concat(schedule_dfs).to_excel(writer)
